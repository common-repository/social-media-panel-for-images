== Description ==

Social Media Panel for Images is plugin to add social buttons on images in posts.
If you want your readers to be able to share post from your blog - this plugin is for you.
You can choose what buttons to show.
Supports next social buttons:

 Twitter
 Facebook
 Google+
 Livejournal
 Youtube
 Vkontakte
 LinkedIn
 Picasa Web Albums
 Flickr
 Pinterest
 DeviantArt

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `social-media-panel-for-images` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `` in your templates
